Instructions

  1. You need to print following logo in a paper or display in a screen first.
    (The marker is known as Kanji Marker)
  2. Then visit this link and allow camera.
  3. Turn on the camera and forcus into the printed marker, and you will see the virtual arena setup on top of the camera video feed.
    Please move away the camera a little if you can't see the environment correctly.
  4. You can use the config panel in the top right corner to adjust the location.
    • Turn on the Coordinate Marker and move the red color dot into any place within given limits
    • Then please refresh the browser to see the updates
    • Yellow color dot indicates the (0,0,0) coordinate.
